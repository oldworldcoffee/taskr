# Quick Start

1. Apply the Supabase migration in `supabase/migrations`.
2. Add env vars from `.env.example` locally and in Vercel.
3. Install dependencies and run `npm run dev`.
4. Sign up or log in.
5. Visit `/setup` or `/restore-super-admin` to grant first super-admin access.
6. Create or self-enroll a company, then add locations and checklists.

For data import/export, use `/super-admin/migration` after super-admin access is active.
